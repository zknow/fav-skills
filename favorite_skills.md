## Productivity

- **[Matt Pocock Skills](https://github.com/mattpocock/skills)** - [[`grill-me`](https://github.com/mattpocock/skills/tree/main/skills/productivity/grill-me)] - 提供防範「憑感覺寫程式」的開發規範技能，最知名的是透過 grill-me 讓 AI 扮演面試官，針對開發計劃逐步提問以理清設計細節。
- **[Anthropic Skills](https://github.com/anthropics/skills)** - [[`docx`](https://github.com/anthropics/skills/tree/main/skills/docx)] [[`pptx`](https://github.com/anthropics/skills/tree/main/skills/pptx)] [[`pdf`](https://github.com/anthropics/skills/tree/main/skills/pdf)] [[`xlsx`](https://github.com/anthropics/skills/tree/main/skills/xlsx)] - Anthropic 官方的 Agent 技能庫，提供強大的文件處理能力，包含 Word (docx), PowerPoint (pptx), PDF 處理與 Excel (xlsx) 試算表分析。

## UI/UX

- **[UI/UX Pro Max](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)** - [[`ui-ux-pro-max`](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill/tree/main/.claude/skills/ui-ux-pro-max)] - AI 驅動的設計智慧技能，提供 UI 風格與顏色搭配規範，協助進行 UI 審查並自動生成設計系統。
- **[Anthropic Skills](https://github.com/anthropics/skills)** - [[`frontend-design`](https://github.com/anthropics/skills/tree/main/skills/frontend-design)] - Anthropic 官方的 Agent 技能庫，最知名的是 frontend-design，能引導 AI 做出大膽且具質感的網頁視覺與動效，避免 AI 特有的死板設計。
- **[Vercel Skills](https://github.com/vercel-labs/agent-skills)** - [[`web-design-guidelines`](https://github.com/vercel-labs/agent-skills/tree/main/skills/web-design-guidelines)] - Vercel 官方的 Agent 技能庫，最知名的是 `web-design-guidelines`，用來審查並優化前端 UI 程式碼，使其符合 100+ 條易用性 (Accessibility), 效能與 UX 規範。

## Code

- **[Vercel Skills](https://github.com/vercel-labs/skills)** - [[`find-skills`](https://github.com/vercel-labs/skills/tree/main/skills/find-skills)] - Vercel 官方的 Agent 技能庫，包含用於搜尋與安裝其它技能的 `find-skills`。
- **[Superpowers](https://github.com/obra/superpowers)** - [[`requesting-code-review`](https://github.com/obra/superpowers/tree/main/skills/requesting-code-review)] - AI 代理人的軟體工程開發規範與技能庫，其中的 `requesting-code-review` 提供結構化的程式碼審查流程，調度審查子代理人對程式碼變更進行多層級評估與回饋。
